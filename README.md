# GIST 604B Assignment 2: QGIS Desktop GIS Projects

## Overview
This repository contains my completed QGIS tutorial projects and outputs for Assignment 2. It demonstrates my ability to use QGIS for core GIS tasks such as map creation, working with coordinate reference systems, editing attribute tables, and performing spatial analysis. The repository includes QGIS project files for each tutorial along with screenshots of the completed results. Overall, this work reflects hands-on experience with foundational GIS workflows in QGIS.

Through these tutorials, I gained experience working with both spatial and tabular data and developed a better understanding of how different GIS tools connect together within a workflow.

## Reflection
This assignment helped a lot of GIS concepts make more sense through actually doing them in QGIS. Working with projections and coordinate systems became much clearer once I could see how they affected my data in real time. I also got more comfortable working with attribute tables and understanding how joins connect different datasets. One of the more challenging parts was making sure all of my layers lined up correctly and using the right tools for each task. Some steps took a few tries, but it got easier as I worked through more tutorials. Overall, I feel much more confident navigating QGIS and completing basic GIS workflows. I can definitely see myself using these skills in future GIS work and projects.
