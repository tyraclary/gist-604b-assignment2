## Project Description
This project focuses on learning the fundamentals of QGIS through a series of tutorials. It includes map creation, working with projections, attribute tables, and performing spatial analysis tasks.

## Tools and Technologies
- QGIS
- OpenStreetMap Data
- Spatial Data (Vector & Raster)

## What I Did
- Created maps using QGIS
- Worked with coordinate reference systems (CRS)
- Performed attribute and spatial joins
- Styled vector and raster layers
- Completed multiple QGIS tutorials

## How to View / Run
Open the QGIS project files (.qgz) using QGIS software.
